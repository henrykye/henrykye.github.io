# henrykye.github.io
This is to host my personal academic website <a href="https://henrykye.com" target="_blank">https://henrykye.com</a>. This is built upon <a href="https://github.com/yenchiah/project-website-template" target="_blank">project-website-template</a> of <a href="http://yenchiah.me" target="_blank">Dr. Yen-Chia Hsu</a>.

**NOTE:** If you want to reuse this website, please make sure that you delete the *Google Analytics* code block in \<head\> on every html page, denoted by `<!-- Global site tag (gtag.js) - Google Analytics -->` and `<!-- End of Google Analytics -->`.
